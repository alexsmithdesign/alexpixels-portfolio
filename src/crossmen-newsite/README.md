# Crossmen Rebrand — Speculative Website

A hypothetical rebrand of the Crossmen Drum & Bugle Corps. Portfolio case study by Alex Smith.

## Stack

Static HTML, CSS, and vanilla JavaScript. No frameworks or build tools.

## File structure

```
src/crossmen-rebrand/
├── index.html          Homepage
├── audition.html       Audition info
├── about.html          The Corps (history, staff, mission)
├── support.html        Club Unity, donations
├── news.html           News card grid
├── shop.html           External store redirect
├── css/
│   ├── reset.css       Box-sizing, margin reset, base
│   ├── tokens.css      Design tokens (colors, type, spacing)
│   ├── layout.css      Container, grid, section spacing
│   ├── components.css  Nav, buttons, cards, footer
│   └── home.css        Homepage-specific styles
├── js/
│   ├── nav.js          Mobile menu, scroll behavior
│   └── main.js         Fade-in observer
└── assets/
    ├── images/         Placeholder — swap with real assets
    └── fonts/          Custom fonts if needed
```

## Disclaimer

This is not affiliated with the Crossmen Drum & Bugle Corps. For demonstration and portfolio purposes only.
